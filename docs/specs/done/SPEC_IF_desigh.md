# Webインターフェース デザインに関する追加要望

- react-i18nextを入れて、英語と日本語の多言語対応にする
- 学習用ユーザ、推論用ユーザについて、R・G・Bそれぞれの商品の「購入閾値」を超えている人数の割合を、グラフで表示する。両者の差分がわかりやすいようにすること
- `Sample Inference Logs` エリアの表示
  - ユーザの「色」情報を表示する。RGBの値と、実際の「色」を表示
  - 提案した個別のアイテムのカラーデザインは、購入が緑・非購入がグレーで表されているが、変更する。
    商品属性R・G・Bに対応し赤・緑・青色とする。また、購入は濃いめの色、非購入は薄めの色で表す。
